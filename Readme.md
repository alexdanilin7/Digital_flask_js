# Тестовое задание для Digital агентство Победа

Тестовое задание предполагает создания простого web приложения на Flask + Vanilla JS.

Для запуска приложения нужно запустить файл 

1. Шаги для запуска приложения с помощью 

    ``docker-compose``

2. Команду для сборки и запуска контейнеров: 

  ``` docker-compose up ```
3. Информацию о том, что приложение будет доступно по адресу `http://localhost:5001`

4. Команду для остановки приложения: 

   ```docker-compose down```
